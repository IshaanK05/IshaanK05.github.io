RF Beamforming Telemetry & Tracking System

Designed and simulated an RF beamforming telemetry and tracking system for small drone platforms, with a focus on electronically steerable antenna arrays to enable directional tracking and robust telemetry links.

Developed a multistage RF front-end and beamforming architecture, including the detailed design and analysis of patch antenna arrays, couplers, phase shifters, attenuators, and hybrid RF components, and evaluated their performance across the 2.4 GHz ISM band.

Modelled and verified the complete RF system using circuit-level and electromagnetic simulation tools (AWR Microwave Office and HFSS), strengthening practical skills in RF system design, antenna theory, beamforming techniques, and high-frequency simulation workflows.

Neural Spikes Analysis Machine Learning Project

This project was a self-initiated exploration aimed at deepening my understanding of machine learning, particularly in the domain of multi-layer perceptrons and unsupervised learning. I developed various ml models using various methods such as clustering, CNNs, and finally MLPs that were capable of filtering electronic noise, detect any spikes in the dataset and then classify those spikes based on what neurons caused them.

The goal was to accurately detect and classify the spikes according to the neuron class that it was caused by. To achieve this, I utilized comprehensive datasets, the first one being a labelled dataset with a low level of electronic noise. And the other datasets being unlabelled with greater levels of noise.

The workflow involved filtering out the electronic noise using methods like bandpass filters or allowing the CNNs and MLPs to train on the noise to be able to distinguish between noise and actual data and then build the detection and classification models using SKlearn and Tensorflow and visualising the training process and results using libraries like numpy, matplotlib.

The project not only reinforced my understanding of neural networks and data augmentation but also gave me practical experience in handling large datasets, optimising model performance, and evaluating classification metrics.

Self Navigating Car

This was another group project that I led, focused on designing and building a battery-powered autonomous car capable of completing a track loop in the shortest possible time. The car navigated by detecting a magnetic field generated by a wire embedded beneath the track, using inductors as sensors.

To achieve this, we developed a complete signal processing and control system, including a band-pass filter, peak detection circuit, motor control unit, and battery management system.

The peak detection circuit extracted directional cues from the magnetic signal and fed this information to an Arduino, which adjusted the car's steering using PWM signals to control each of the motors.

Leading this project strengthened my skills in analog circuit design, embedded systems, and sensor integration. It also enhanced my ability to manage a multidisciplinary team and coordinate efforts across hardware and software development.

Dual Wavelength Pulse Oximeter

This project involved designing and building a dual-wavelength pulse oximeter from the ground up. I assembled the electronic circuitry by soldering various components onto a custom PCB using multiple soldering techniques.

The circuit was then connected to an Arduino microcontroller. The device used photodiodes to detect light absorption at two wavelengths, allowing it to measure both heart rate and blood oxygen saturation (SpO₂).

Real-time heart rate data was visualized using the Arduino IDE's serial plotter. The oximeter was tested extensively across multiple users to ensure accuracy and reliability.

This project significantly enhanced my skills in circuit assembly, sensor integration, embedded programming, and physiological signal processing.
